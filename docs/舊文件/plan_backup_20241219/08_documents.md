# Module 08：Documents (文件庫與向量化)

## 1. 模組目標
集中管理專案的所有檔案（規格書、會議記錄、截圖）。為未來的 RAG (語義搜尋) 提供結構化基礎，並支援文件與核心物件 (Decision/Action) 的掛載關聯。

## 2. 功能需求 (Requirements)

### 2.1 文件庫管理
- **類型分類**：規格書 (Spec)、會議記錄 (Meeting)、設計 (Design)、其他。
- **解析與預覽**：支援 PDF/Word 目錄解析 (樹狀導航) 與基礎預覽。
- **關聯檢視**：顯示該文件衍生出了哪些 Action / Decision。

### 2.2 語義處理 (基礎)
- **自動摘要**：上傳時透過 AI 產生文件內容摘要。
- **引用標記**：支援將文件中的特定章節/段落作為 Citation 引用。

## 3. 驗收標準
1. 文件可依類型篩選。
2. 點擊文件可看到由其產生的所有建議卡歷史。
3. 支援章節樹狀結構展示（若文件具備層級）。
