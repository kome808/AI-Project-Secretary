# 模組：文件庫（Sources / Artifacts）

## 目標
管理所有來源（Artifacts），提供「反向追蹤」：一份來源衍生了哪些任務/待回覆/變更/決議/專案工作。

## UI 需求（MVP）
- 清單：類型（文字/對話/連結/檔案/圖檔）、標題、建立時間、引用數
- 詳情：內容檢視 + 引用清單（按類型分組）+ 點擊跳轉項目詳情
- 原則：來源內容不可修改（可改名稱/封存）
- 篩選器：
  - 類型篩選：全部 / 文字 / 對話 / 連結 / 檔案 / 圖檔
  - 引用篩選：全部 / 已引用 / 未引用
  - 搜尋：文件內容或標題關鍵字

## 檔案類型說明
- **文字**：純文字內容（text/plain）
- **對話**：對話記錄（text/conversation）
- **連結**：URL 連結（text/uri-list）
- **檔案**：PDF、Word、Excel 等文件檔案（application/*，排除圖片）
- **圖檔**：所有圖片格式（image/*）

## 驗收標準
- 可從任一項目的 Citation 點回來源
- 可從來源看到所有衍生項目
- 可透過類型篩選快速找到特定格式的文件
- 可搜尋文件內容或標題
