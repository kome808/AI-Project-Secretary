# 模組：儀表板（Dashboard）

## 目標
用「虛擬秘書」口吻提供今日焦點：需要你處理、風險提示、快速詢問入口。

## 主要情境
1) 早上進來：秘書用簡報卡說「今天最重要的 3 件事」
2) 快速知道：哪些逾期/卡關/待回覆
3) 直接問：用 RAG 查「上次決議是什麼」

## UI 需求（MVP）
- 首屏只保留 3 區塊（避免資訊爆炸）：
  1) **晨間簡報卡**（Summary + Risks + Top Actions + Citations）
  2) **需要你處理**（列表：逾期/卡關/待回覆混合，最多 8 筆）
  3) **AI 入口**（醒目輸入列 + 提示詞 chips）
- 每筆建議/提醒必有 Citation 或可跳到相關項目詳情

## 驗收標準
- 能看到晨間簡報卡（Mock 可）
- 需要你處理列表可點入任務/待回覆/變更詳情
- AI 入口可以送出並產生 Inbox 建議卡或觸發 RAG 搜尋

## 測試案例
- 有 2 筆逾期任務 → 儀表板顯示於需要你處理
- 有 1 筆等待客戶回覆 7 天 → 顯示待回覆提示
- 點擊 Citation → 可跳到來源 Artifact
