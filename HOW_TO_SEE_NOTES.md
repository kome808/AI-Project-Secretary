# 🔍 如何查看備註功能

## 方法 1️⃣：在「新增/編輯任務」對話框中

### 📍 位置
當您新增或編輯任務時，對話框中會有一個「備註」欄位。

### 🎯 步驟
```
1. 前往「任務清單」→「專案工作」頁面
   
2. 點擊任何工作包卡片上的「➕ 新增任務」按鈕
   （如果沒有工作包，先點擊右上角的「➕ 新增工作包」建立一個）

3. 在彈出的對話框中，您會看到以下欄位（從上到下）：
   ┌─────────────────────────────────┐
   │ ✏️ 編輯任務                      │
   ├─────────────────────────────────┤
   │ 名稱 *                          │
   │ [輸入任務名稱]                   │
   │                                 │
   │ 描述                            │
   │ [詳細說明此任務的內容]           │
   │                                 │
   │ 狀態                            │
   │ [待處理 ▼]                      │
   │                                 │
   │ 負責人                          │
   │ [選擇負責人 ▼]                  │
   │                                 │
   │ 期限                            │
   │ [選擇日期]                       │
   │                                 │
   │ ⭐ 備註（選填）⭐                │
   │ [在此輸入備註或補充說明...]      │  ← 👈 這裡就是備註欄位！
   │                                 │
   │ X 個字元                        │
   │                                 │
   │         [取消]  [儲存]          │
   └─────────────────────────────────┘

4. 在「備註」欄位中輸入文字，例如：
   "這是測試備註
    可以輸入多行內容"

5. 點擊「儲存」按鈕
```

### ✅ 確認成功
儲存後，對話框會關閉，您剛建立/編輯的任務現在就有備註了！

---

## 方法 2️⃣：在任務卡片中查看/編輯備註

### 📍 位置
在任務卡片**展開**後的底部區域。

### 🎯 步驟
```
1. 找到您剛才建立的任務卡片（或任何現有任務）

2. 任務卡片左側有一個「>」圖示，點擊它來展開任務
   
   收起狀態：
   ┌─────────────────────────────────┐
   │ ▶ 任務名稱    負責人  [狀態]    │  ← 點擊左側的「▶」
   └─────────────────────────────────┘

3. 展開後，卡片會變成這樣：
   ┌─────────────────────────────────┐
   │ ▼ 任務名稱    負責人  [狀態]    │
   ├─────────────────────────────────┤
   │ 詳細說明內容...                  │
   │                                 │
   │ 更新於 2024/12/21               │
   │                                 │
   │ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │  ← 分隔線
   │                                 │
   │ 💬 備註                         │  ← 👈 這裡！
   │ ┌─────────────────────────────┐ │
   │ │ 這是測試備註                 │ │
   │ │ 可以輸入多行內容             │ │
   │ │                             │ │
   │ │ 最後更新：5 分鐘前 by 你     │ │
   │ └─────────────────────────────┘ │
   └─────────────────────────────────┘

4. 點擊備註區域可以直接編輯
   
5. 編輯模式：
   ┌─────────────────────────────────┐
   │ 💬 編輯備註                      │
   │ ┌─────────────────────────────┐ │
   │ │ [可編輯的文字框]             │ │
   │ │                             │ │
   │ └─────────────────────────────┘ │
   │ [✓ 儲存] [✗ 取消]               │
   │ 提示：Ctrl+Enter 儲存，Esc 取消  │
   └─────────────────────────────────┘

6. 快捷鍵：
   - Ctrl+Enter（或 Cmd+Enter）：儲存
   - Esc：取消編輯
```

## ❓ 疑難排解

### Q: 我看不到備註欄位
**A**: 請確認：
- ✅ 您打開的是「新增/編輯任務」對話框（不是「新增工作包」）
- ✅ 對話框可能需要向下滾動才能看到備註欄位（在「期限」下方）

### Q: 任務卡片上看不到備註
**A**: 請確認：
- ✅ 您已經點擊任務卡片左側的「>」圖示展開任務
- ✅ 備註區域在展開區域的最底部（在「更新於」下方）
- ✅ 該任務確實有備註內容（如果沒有會顯示「點擊新增備註」）

### Q: 我建立的任務沒有儲存備註
**A**: 請確認：
- ✅ 您在對話框的「備註」欄位中確實輸入了文字
- ✅ 點擊了「儲存」按鈕（而不是直接關閉對話框）

## 🎨 支援備註的所有位置

備註功能已整合到以下所有任務相關的地方：

1. **編輯對話框**：ItemEditDialog（用於新增/編輯任務）
2. **專案工作視圖**：CompactItemCard（展開後可見）
3. **待回覆/需求變更視圖**：ExpandableItemCard（展開後可見）
4. **決議視圖**：ItemCard（卡片底部）

## 📸 測試建議

1. 建立一個新任務，在備註欄位輸入：
   ```
   這是第一行備註
   這是第二行備註
   這是第三行備註（會被折疊）
   ```

2. 儲存後，展開任務卡片

3. 您應該會看到前兩行備註，以及一個「展開全文」按鈕

4. 點擊「展開全文」可以看到完整內容

5. 點擊備註區域可以直接編輯
