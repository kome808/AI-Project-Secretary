# 測試資料初始化指南

## 快速開始

1. 前往「設定」頁面（Settings）
2. 找到「開發工具：資料管理」面板（橙色邊框）
3. 點擊「建立『國美館台灣美術知識庫』測試資料」按鈕
4. 等待提示訊息，然後重新整理頁面

## 清除資料

如需清除所有資料並重新開始：

1. 前往「設定」頁面
2. 在開發工具面板中點擊「清除所有資料」
3. 再次點擊確認
4. 重新整理頁面

## 注意事項

- 此功能僅供開發測試使用
- 清除資料會移除 localStorage 中的所有內容
- 建立測試資料會自動設定當前用戶為「王雅婷（PM）」
- 生產環境請移除 DevToolsPanel 元件
